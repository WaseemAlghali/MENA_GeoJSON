# MENA_GeoJSON

- Bahrain
- Djibouti

- Syria
- Tunisia
- United Arab Emirates
- Palestine
- Yemen


## Project Structure
- countries: directory of the countries GeoJSON files
- regions: directory of the regions GeoJSON data of each country
